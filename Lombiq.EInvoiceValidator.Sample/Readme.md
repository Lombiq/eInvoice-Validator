# Lombiq eInvoice Validator Sample

## About

This is a sample console application to demonstrate how to use the Lombiq eInvoice Validator library. It shows how to validate EN 16931 CII and UBL invoices. The samples are from [CEN/TC 434 - EN-16931 - Validation artifacts](https://github.com/ConnectingEurope/eInvoicing-EN16931) repository.

## How to use

Run the project from your IDE.

## Sample Invoices

The sample invoices are located in the `SampleInvoices` folder. They include both CII and UBL formats. If you want to update them, check the latest [CEN/TC 434 - EN-16931 - Validation artifacts](https://github.com/ConnectingEurope/eInvoicing-EN16931) repository for the latest samples.
